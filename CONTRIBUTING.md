# Contributing to Lost in Hyd

Thank you for your interest in contributing to Lost in Hyd!

## How to Contribute

### Submit a Hidden Gem

Know a great place in Hyderabad that more people should know about? We'd love to feature it!

**Open an issue** with:
- Name of the place
- Location/Address
- What makes it special
- Best time to visit
- Your name (for credit)

### Report Corrections

Found outdated info? Let us know by opening an issue.

### Suggest Topics

Have an idea for a neighborhood deep-dive or theme? Share it!

## Content Guidelines

- Focus on Hyderabad and surrounding areas
- Prioritize local, independent businesses
- Include practical details (timings, location, approx costs)
- Respect privacy - ask before featuring individuals
- Verify information before submitting

## Code Contributions

For technical contributors:
- Fork the repo
- Create a feature branch
- Submit a PR with clear description
