# Lost in Hyd - Project Structure

```
lostinhyd/
├── README.md              # Project overview
├── LICENSE                # License info
├── CONTRIBUTING.md        # How to contribute
├── newsletters/           # Draft newsletter content
│   ├── draft.md
│   └── template.md
├── editions/              # Published newsletter editions
│   ├── 2025/
│   │   ├── 001-welcome.md
│   │   └── 002-charminar-food-walk.md
│   └── archive/
├── assets/                # Images, logos, etc.
│   ├── logo/
│   ├── photos/
│   └── graphics/
├── templates/             # Email/Newsletter templates
│   ├── email-template.html
│   └── markdown-template.md
└── scripts/               # Automation scripts
    ├── send-newsletter.py
    └── scrape-events.py
```

## Getting Started

1. Draft content in `newsletters/draft.md`
2. Move to `editions/YYYY/NNN-title.md` when published
3. Use templates in `templates/` for consistent formatting
4. Store images in `assets/photos/` with descriptive names
